# datapi
Exposition des jeux de données


## Python

`cd python`

### Run

`fastapi dev` or in pord `fastapi run`

See :
* Homepage: http://127.0.0.1:8000/
* Swagger Documentation: http://127.0.0.1:5000/docs/atlas2023
* OpenAPI JSON format: http://127.0.0.1:5000/json/atlas2023


## Javascript

`cd js`

### Installation

`npm i`


### Run

`node server.js`

Pour la collection "atlas2023" :
* Swagger Documentation: http://localhost:3000/doc/atlas2023
* OpenAPI JSON format: http://localhost:3000/api/doc/atlas2023
* API: http://localhost:3000/api/atlas2023
